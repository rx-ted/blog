# hexo blog


## hexo new 
创建文章如下命令：
```bash
hexo new example

```
就会创建example.md,在`./source/_post/example.md`



## hexo g 
生成静态源代码网页
创建路径`./public/`

## hexo d
上传github仓库，并同步github pages.

## 保存源代码标记tag为X.X.X
```bash
git tag X.X.X
git push origin X.X.X
```
