## 📦 Project Commands

Choose your package manager and run the commands below:

:::code-group

```bash [npm]
# 🧲 Clone the project
git clone https://github.com/rx-ted/blog.git
git switch vitepress
git pull
cd blog

# 📦 Install dependencies
npm install

# ▶️ Start development server
npm run dev

# 🏗️ Build for production
npm run build
````
