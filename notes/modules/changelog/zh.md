## 📦 项目运行命令

请选择你的包管理器执行对应命令：

:::code-group

```bash [npm]
# 🧲 克隆项目
git clone https://github.com/rx-ted/blog.git
git switch vitepress
git pull
cd blog

# 📦 安装依赖
npm install

# ▶️ 启动本地开发
npm run dev

# 🏗️ 构建生产版本
npm run build
````
